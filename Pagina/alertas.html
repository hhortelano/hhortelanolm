Para una alerta hay que poner varias clases, la principal es 'alert' para cambiar de color tienes las clases 'alert-primary(o otros)'

Luego para poder cerrarlas tienes que poner las clases 'alerts-dismissible' y crear un boton con clase 'btn-close' luego un aria-label="close" y por utlimo data-bs-dismiss="alert"

Por ultimo puedes poner tambien links a otros sitios en la alerta con un 'a' con clase alert-link
